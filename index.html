Loading Box Widget... using Materialize

Whe you work with Materialize you can use the Preloader; if you only use it once is not a problem, but what about using it more times?... you must put the same code in each of your web pages, showing and hiding it.

LoadingBox helps you showing loading message easier.

How to use it?

just adding the loading-box.css and loading-box.js after the Materialize links and then you can call the box wherever you want using this:

Materialize.LoadingBox.open("Loading Message");

Select a demo clicking in a item on the sideNav

Code:


    Materialize.LoadingBox.open("Loading Message");

Code:


    Materialize.LoadingBox.open('Loading message...', {color:'flashing'});

Code:


    Materialize.LoadingBox.open(
        'Loading message...',
        {
            type: 'determinate',
            progress: 50
        }
    );

Code:


    Materialize.LoadingBox.open(
        'Loading...',
        {
            type: 'determinate',
            progress: 1
            closeOnFinishProgress: true,
            position: 'top_right'
            }
        );

        // demo update progress
        var x = 1;
        setInterval(
            function () {
                if (x <= 100) {
                    Materialize.LoadingBox.updateText("loading... "+x+"%");
                    Materialize.LoadingBox.updateProgress(x++);
                }
            },
            100
        );

Code:


    Materialize.LoadingBox.open(
        'Loading elements...',
        {
            type:'indeterminate',
            color: 'red'
        }
    );
